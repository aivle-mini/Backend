# AIVLE mini project
## 🖥️ 프로젝트 소개
책의 내용에 따라 책의 표지를 자동으로 생성하여 책을 기록하는 도서 관리 시스템입니다.

### ⚙️ 개발 환경
- `Java 17.0.12`
- `JDK 17.0.12`
- **Framework** : Springboot(3.5.0)
- **Database** : H2
- **Front-end** : 리액트
